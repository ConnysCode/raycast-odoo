# Odoo - Time Keeper

Track time, manage timesheets, and monitor attendance in Odoo.

## Installation

1. Clone the repository
2. Run `yarn` to install dependencies
3. Open Raycast and run the **Import Extension** command
4. Select the cloned repository folder
5. Run `yarn dev` once to build and register the extension â€” you can stop it after the extension appears in Raycast
